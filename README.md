# Marathoner: Walk Around The World !

## Introduction
Nowadays, staying physically active has become increasingly difficult. Social media, streaming platforms, and digital entertainment encourage people to spend more time sitting and less time moving. That is why I wanted to create **Marathoner**, a web application that motivates people to do physical activities, while exploring the world virtually.

## Instructions
Here are some instructions for running the project.

First, clone the repository on the local device.
```
  git clone git@github.com:Cospiz67/marathoner.git
```
Then, move into the cloned directory.
```
  cd marathoner
```
After that, make sure node is installed on your computer and run the following command:
```
  npm install
```
Finally, to run the web application, run the command:
```
  npm run
```
